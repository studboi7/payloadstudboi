
# PREREQUISITE

  


Install build-essential by the command:

```bash
    sudo apt install build-essential
    sudo apt install g++
```


## Installation


```bash
   git clone https://github.com/studboi7/payloadstudboi.git 
   cd payloadstudboi
   mv conio.h /usr/include/
   g++ payload.cpp
   ./a.out
```
